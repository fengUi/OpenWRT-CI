# Athena Stable Build

This branch is rebuilt from `VIKINGYFY/OpenWRT-CI` for JDCloud AX1800 Pro / `jdcloud_re-ss-01`.

It keeps the build minimal and pins the Lucky runtime to the locally verified `Lucky 3.0.0 wanji` binary in `Files/usr/libexec/lucky-3.0.0-wanji`.

The firmware does not include broadband account credentials.
